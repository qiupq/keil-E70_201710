00030 /**
00031  *  \page gmac GMAC Example
00032  *
00033  *  \section Purpose
00034  *
00035  *  This example uses the Gigabit Ethernet MAC (GMAC) and the on-board Ethernet
00036  *  transceiver available on Atmel evaluation kits. It enables the device to
00037  *  respond to a ping command sent by a host computer.
00038  *
00039  *  \section Requirements
00040  *
00041  * This package can be used with SAMV71 Xplained Ultra board or SAME70 Xplained board.
00042  *
00043  *  \section Description
00044  *
00045  *  Upon startup, the program will configure the GMAC with a default IP and
00046  *  MAC address and then ask the transceiver to auto-negotiate the best mode
00047  *  of operation. Once this is done, it will start to monitor incoming packets
00048  *  and processing them whenever appropriate.
00049  *
00050  *  The basic will only answer to two kinds of packets:
00051  *
00052  *  - It will reply to ARP requests with its MAC address,
00053  *  - and to ICMP ECHO request so the device can be PING'ed.
00054  *
00055  *
00056  *  \section Usage
00057  *
00058  *  -# Build the program and download it inside the board.
00059  *     Please refer to the Getting Started with SAM V71/E70 Microcontrollers.pdf
00060  *  -# On the computer, open and configure a terminal application
00061  *     (e.g. HyperTerminal on Microsoft Windows) with these settings:
00062  *    - 115200 baud rate
00063  *    - 8 bits of data
00064  *    - No parity
00065  *    - 1 stop bit
00066  *    - No flow control
00067  *  -# Connect an Ethernet cable between the evaluation board and the network.
00068  *      The board may be connected directly to a computer; in this case,
00069  *      make sure to use a cross/twisted wired cable such as the one provided
00070  *      with the evaluation kit.
00071  *  -# Start the application. It will display the following message on the DBGU:
00072  *     \code
00073  *      -- GMAC Example xxx --
00074  *      -- SAMxxxxx-xx
00075  *      -- Compiled: xxx xx xxxx xx:xx:xx --
00076  *      MAC 3a 1f 34 08 54 54
00077  *      IP  192, 168, 1, 3
00078  *     \endcode
00079  *  -# The program will then auto-negotiate the mode of operation and start
00080  *     receiving packets, displaying feedback on the DBGU. To display additional
00081  *     information, press any key in the terminal application.
00082  *  \note
00083  *  Make sure the IP address of the device(EK board) and the computer are in the
00084  *  same network.
00085  *
00086  *  \section References
00087  *  - gmac/main.c
00088  *  - gmacb.h
00089  *  - gmacd.h
00090  *  - gmac.h
00091  */
