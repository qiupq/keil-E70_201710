SAM E70 Xplained - Board

Description

Purpose

This file is dedicated to describe the SAM E70 Xplained board.

Contents

  • For SAM E70 Xplained board information, see "SAM E70 Xplained - Board informations".
  • For operating frequency information, see "SAM E70 Xplained - Operating frequencies".
  • For using portable PIO definitions, see "SAM E70 Xplained - PIO definitions".
  • For using GMAC PIO definitions, see "SAM E70 Xplained - GMAC".
  • For using ISI definitions, see "SAM E70 Xplained - ISI".
  • For on-board memories, see "SAM E70 Xplained - Memories".
  • Several USB definitions are included here, see "SAM E70 Xplained - USB device".
  • For External components, see "SAM E70 Xplained - External components".
  • For Individual chip definition, see "SAM E70 Xplained - Individual chip definition".

To get more software details and the full list of parameters related to the SAM E70 Xplained board configuration, please have a look at the source file: board.h, board_e70_xpld.h

Usage

  • The code for booting the board is provided by startup_same70.c and board_lowlevel.c.
  • For using board PIOs, board characteristics (clock, etc.) and external components, see board.h, board_e70_xpld.h.
  • For manipulating memories, see board_memories.h.

This file can be used as a template and modified to fit a custom board, with specific PIOs usage or memory connections.
